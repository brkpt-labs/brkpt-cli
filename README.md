# @brkpt/cli

The official CLI for brkpt.

## Installation

```bash
npm install -g @brkpt/cli
```

## brkpt auth

Authentication scaffolding for NestJS, powered by [brkpt-auth](https://github.com/brkpt-labs/brkpt-auth).

```bash
# Initialize brkpt-auth in your project
brkpt auth init

# Add a feature
brkpt auth add

# Add a feature with a specific verifier
brkpt auth add oauth --verifier google
```
